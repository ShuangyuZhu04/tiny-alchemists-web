My Contributions

💠

Lead Game Design

I designed the core systems of the game, including the game loop, potion crafting mechanics, player abilities, and miasma system. I led discussions on game design with other team members and incorporated their ideas and feedback. I was also responsible for maintaining the main parts of the Game Design Document to precisely communicate the implementation requirements.

💠

System Architecture

I set up and maintained the structure of the game code and assets, ensuring that the repository was cohesively organized. I created the modular architecture of the Unity C# scripts to keep functionality encapsulated for flexible collaboration and easier feature updates.

💠

General Programming

I coded a significant portion of the game systems: flexible ingredient type and modifier system for brewing combined potion effects; cauldron and planting logic; sound and particle effect integration; player and object status effects; dynamically growing miasma on a voxel grid.

💠

Multiplayer Programming

I contributed to the multiplayer integration, which involved spawning players who join a session, resolving ownership across the network, as well as syncing physics and behaviour logic for each client.
